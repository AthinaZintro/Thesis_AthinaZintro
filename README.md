#Thesis_AthinaZintro

## Περιγραφή
Το project παίρνει αρχεία (.csv) και εντολές query "Select from where" και αποθηκεύει τα αποτελέσματα του σε ένα αρχείο (.txt).


##Εγκατάσταση 
1. Εγκατάσταση Java. 
2. Εγκατάσταση βιβλιοθήκων javax.swing και java.awt για να χρησιμοποιηθεί το Gui.

##Χρήση

Το project μπορεί να εκτελεστεί με διάφορους τρόπους.

1) Μπορεί να εκτελεστεί απο τον clientGuiApplication που είναι η απεικόνιση σε Gui του project και όλων των ιδιοτήτων του.
 
 * Εκτέλεση κλάσης Application.
 * Επιλογή της λειτουργίας "Register File",γιατί κάθε αρχείο που θέλουμε να εκτελέσουμε σε αυτό μια εντολή query πρέπει να είναι καταγεγραμμένο.Αν έχει καταγραφεί σε προηγούμενη χρήση του project τότε μπορούμε να περάσουμε στην επιλογή του "Query Execution".
 * Μετα τη επιλογή της "Register File", ανοίγει νέο παράθυρο που δίνει δυο επιλογές καταγραφής αρχείου στον χρήστη την "Choose File" και την "Give information".
 * Στην "Choose File" ανοίγει ένα παράθυρο διαλέγουμε ποιο αρχείο θέλουμε να καταγράψουμε και δημιουργείται αυτόματα το αρχείο καταγραφής "ΌνομαΑρχείου"Metadata.txt.Προσοχή!!! Τα αρχεία πρέπει να είναι της μορφής (.csv) και στις 4 πρώτες γραμμές να περιέχουν το όνομα του αρχείου με την επέκταση(.csv),τον οριοθέτη που χωρίζει τις στήλες,τον αριθμό των στηλών, και στην τέταρτη γραμμή όλες τις στήλες χωρισμένες με τον οριοθέτη τους από την πέμπτη γραμμή και μετά αρχίζουν τα δεδομένα του αρχέιου.
 * Στην "Give information" δίνουμε τις απαράιτητες πληροφορίες για να γίνει η καταγραφή του αρχείου πατώντας "Register File" δημιουργείται αυτόματα το αρχείο καταγραφής "ΌνομαΑρχείου"Metadata.txt. Προσοχή!!! Τα στοιχεία που θα γραφτούν πρέπει να γραφτούν όπως είναι στο αρχείο. Το όνομα του αρχείου γράφεται με την επέκταση(.csv) και οι στήλες με την σειρά που είναι στο αρχείο.
 * Μετα τη επιλογή της "Query Execution", ανοίγει νέο παράθυρο που δίνει την δυνατότητα στον χρήστη να γράψει το query και να το εκτελέσει.Προσοχή!!! Τα στοιχεία που έχουν να κάνουν με το αρχείο πρέπει να γραφτούν όπως είναι στο αρχείο, το όνομα του αρχείου χωρίς την επέκταση(.csv).Τα στοιχεία της εντολής όπως "Select","from" και "where" μπορούν να γραφτούν είτε με κεφαλαία είτε με μικρά είτε συνδυασμό κεφαλαίων μικρών. Μόλις πατηθεί το "Execute Query" τα αποτελέσματα του query αποθηκεύει τα αποτελέσματα σε ένα νέο αρχείο "ΌνομαΑρχείου"Results.txt. 
 * Κάθε παράθυρο έχει το κουμπί "<< Back" που μας δίνει την δυνατότητα να πάμε πίσω στο προηγούμενο παράθυρο.
 
2) Μπορεί να εκτελεστεί απο τον clientRegisterApplication που μας δίνει την δυνατότητα μέσω του terminal να περάσουμε τα απαραιτητα στοιχεία ως "args[]",δηλαδή το όνομα του αρχείου με την επέκταση(.csv),τον οριοθέτη που χωρίζει τις στήλες,τον αριθμό των στηλών, και τις στήλες χωρισμένες με κενό για να γίνει η καταγραφή του αρχείου και να δημιουργηθεί το αρχείο καταγραφής "ΌνομαΑρχείου"Metadata.txt.

3) Μπορεί να εκτελεστεί με απο τον clientQueringApplication που μας δίνει την δυνατότητα μέσω του terminal να περάσουμε τα απαραιτητα στοιχεία ως "args[]",σε αυτή την περίπτωση το query και αν εχει γίνει η καταγραφή του αρχείου να εκτελέσει το query και να αποθηκεύσει τα αποτελέσματα σε ένα νέο αρχείο "ΌνομαΑρχείου"Results.txt.
